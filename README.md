# 沉浸式学习计时器 (Immersive Study Timer)

![screenshot](./document/anim.gif)

一款专为对抗“计时器焦虑”而设计的桌面学习伴侣。它通过随机学习时长与自动长短休息机制，帮助你进入心流状态，让每一次专注都充满未知感和成就感。

## 源起与理念

本项目的创作灵感完全来自于 Bilibili UP主 **择恩** 的视频分享。视频中提出的“比番茄钟更好用的随机计时器”理念，完美解决了传统番茄工作法带来的“计时器焦虑”问题。

为了将这个优秀的理念付诸实践，并方便自己使用，我开发了这款桌面小工具。

> **核心理念来源**: [为什么我能每天学习10小时 - 择恩](https://www.bilibili.com/video/BV1naLozQEBq/?spm_id_from=333.1391.0.0&vd_source=ba468568caebc92479698e83c28be8b0)

## 核心特性

*   🎲 **随机周期**: 在设定的范围内随机生成学习时长，打破固定周期的枯燥。
*   ☕️ **智能休息**: 自动在短休息和长休息之间切换，确保劳逸结合。
*   🏅 **胜利成就**: 大专注完成后播放 `victory.mp3` 音效，增强成就感。
*   📊 **精美统计**: 自动按天汇总汇总学习记录，生成卡片式 HTML 统计报表，支持复盘。
*   🗄️ **SQLite 存储**: 采用轻量级数据库存储，数据更安全、更易管理。
*   🖱️ **鼠标穿透**: 可将窗口“锁定”，使其不响应任何鼠标事件，避免误触，不遮挡下方内容。
*   ⌨️ **全局热键**: 无需激活窗口，即可通过快捷键控制计时器的开始、暂停与重置。
*   🎨 **高度定制**: 通过 `config.json` 文件，可自由修改时间、声音、快捷键等参数。

## 快速开始 (推荐给普通用户)

1.  前往本项目的 [**Releases**](https://github.com/Yogioo/StudyTimer/releases) 页面。
2.  下载最新的 `study_timer_Vx.x.7z` 压缩包。
3.  解压后，确保 `study_music` 文件夹中包含必要的音频文件，然后双击运行 `study_timer_gui.exe`！

## 使用指南

*   **移动与缩放**: 在“解锁”状态下，通过顶部区域拖动窗口，或使用右下角手柄缩放。
*   **锁定/解锁**: **双击窗口** 或通过右键菜单切换。锁定后，窗口将对鼠标“透明”。
*   **结束休息**: 在长休息期间，界面会显示“结束休息”按钮，点击可提前返回工作。

#### 右键菜单功能

| 功能 | 说明 |
| :--- | :--- |
| **状态信息** | 显示当前计时器状态和距离下一次长休息的剩余时间。 |
| **▶️ 开始/继续** | 启动计时器。手动开始时无噪音，短休息结束恢复时有提示音。 |
| **⏸️ 暂 停** | 暂停当前计时，需输入暂停理由以便统计。 |
| **📊 查看统计** | **(核心更新)** 用浏览器打开按天分组的卡片式统计报表。 |
| **⚙️设置** | 快速调整随机间隔、大专注时长触发点及修改快捷键。 |
| **💧 透明度** | 调整窗口透明度。 |
| **🔄 重 置** | 提供重置当前轮次或清空所有历史数据（需授权密码）。 |
| **❌ 退 出** | 保存所有进度并退出。 |

#### 全局快捷键 (默认)

| 操作 | 快捷键 |
| :--- | :--- |
| **开始 / 继续** | `Alt` + `Z` |
| **暂停 / 恢复** | `Alt` + `C` |
| **重置当前轮次** | `Ctrl` + `Alt` + `R` |

> 以上快捷键可以在 `config.json` 或通过右键菜单直接修改。

## 自定义配置 (`config.json`)

程序首次运行后会自动生成 `config.json`。修改后重启程序生效。

| 参数 | 说明 |
| :--- | :--- |
| `study_time_min` | 每轮学习的最短时间 (秒)。 |
| `study_time_max` | 每轮学习的最长时间 (秒)。 |
| `long_break_threshold`| 触发长休息所需的累计学习总时长 (秒)。 |
| `reset_password` | 执行清空记录等高危操作的授权密码。 |

```bash

```

#### 3. 构建可执行文件
根目录提供脚本:
- `clear_then_build.bat`: 一键清理旧文件并使用 PyInstaller 打包。
